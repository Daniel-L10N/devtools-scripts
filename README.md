# DevTools & Scripts 🚀

Essential scripts and tools for developers.

## Features

- Productivity boosters
- Automation scripts
- CLI utilities
- API helpers

## Quick Start

```bash
# Clone the repo
git clone https://github.com/Daniel-L10N/devtools-scripts.git

# Run a script
node scripts/cleanup.js
```

## Scripts Included

| Script | Description |
|--------|-------------|
| `cleanup.js` | Clean up node_modules |
| `backup.js` | Auto-backup your files |
| `deploy.js` | Deploy to multiple platforms |
